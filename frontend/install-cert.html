Certificat SEDI-ATI

Scanne le QR code avec ta tablette pour télécharger et installer le certificat

📱 Installation Android

1
Scanne le QR code ou appuie sur le bouton
2
Le fichier SEDI-ATI-CA.crt se télécharge
3
Paramètres → Sécurité → Installer un certificat → Certificat CA
4
Sélectionne le fichier téléchargé et accepte
⬇️ Télécharger le certificat

Ce certificat supprime l'avertissement de sécurité sur le site SEDI-ATI.
Valide 10 ans — ne donne accès à aucune donnée personnelle.
